# Agent Harness Skills For Gemini

This extension provides reusable skills for repository harness design.

Use `INDEX.md` to choose the right skill before answering repository harness questions. The skills live under `skills/<skill-name>/SKILL.md` and cover agent entrypoints, repository boundaries, validation surfaces, runtime evidence, delivery records, quality gates, design docs, task boards, and atomic commits.

Keep skill usage repo-neutral. Do not copy private environment assumptions, URLs, credentials, or project-specific workflow details into reusable guidance.
